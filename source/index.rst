概述
======

焊接 SDK 是一套用于控制焊接机器人执行多种摆焊轨迹的编程接口。提供了摆焊参数配置、实时控制、状态反馈功能，支持 Z字摆、三角摆、台形摆、月牙摆、正弦摆、圆形摆、8字摆，适用于自动化焊接生产线、机器人焊接工作站等工业场景。

本文档介绍了JAKA焊接工艺包专用SDK接口的规格和使用案例。
